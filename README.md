# Block Story Mods

A set of quality-of-life and content mods for Block Story, built on the **BlockStoryCore** framework. They never touch the game's own files.

The recommended way to install and keep these updated is the [Block Story Mod Kit](https://github.com/1MR1C1/BlockStoryModKit) — it sets up your game, installs the framework, and manages the mods for you.

## The mods

- **Minimap** — round, depth-aware minimap, a fullscreen world map, and waypoints. Toggle with the Minimap key.
- **Vein Miner** — break a whole vein (or a set shape) in one hit. The loot goes straight to your inventory.
- **Xray** — see ores, valuable blocks and liquids through walls. Toggle with the Xray key.
- **Character Editor** — live-edit your stats, vitals, level, XP and skill points. Open with the Character key.
- **NPC Menu** — talk to any NPC from anywhere to take or hand in quests. Open with the NPC key.
- **Xenor** — adds the Xenor, a detailed xenomorph creature that spawns in the space biomes. Catch one with a soul catcher, craft its spawner and soul, and raise your own as a rideable pet.

All of these run on the **BlockStoryCore** framework, which the [Block Story Mod Kit](https://github.com/1MR1C1/BlockStoryModKit) installs for you.

## Install

1. Install the [Block Story Mod Kit](https://github.com/1MR1C1/BlockStoryModKit) and use it to set up your game for mods.
2. Add the mods you want through the Mod Kit. You can also grab the `.dll` files from the [Releases](../../releases) page and install them with the Kit.
3. Start the game. Each mod's hotkey can be changed in the in-game keybinding menu.

The compiled mods are also in the `mods/` folder of this repository.
